              Make sure myDATA and Softone integrations are activated in your Elorus account before using the guide bellow.
              The guide thinks you are a new Elorus user, so some steps can be omitted eg "Create Contact" etc.

              Authorization variables:

              • Token: API key. Find it in Elorus UI "User Profile". One API key per Elorus account
              • X-Elorus-Organization: Organization ID. Find it in Elorus UI "Settings -> Organization -> Organization ID". One Organization ID per Organization.

              Create a "Simple Invoice" - without myDATA integration

              1.1 Create a new contact

              First lets create a contact so there is someone to bill.
              Request
              $ curl -X POST -H "Content-Type: application/json" -H "Authorization: Token ~Your-API-key~" -H "X-Elorus-Organization: ~Your-Organization-ID~" https://api.elorus.com/v1.1/contacts/ -d '{"client_type": "1", "company":"Elorus SA", "vat_number":"9876543210", "addresses":[{"address":"Theofilopoulou 13", "city":"Athens", "zip":"11743", "country": "GR", "ad_type": "bill"}], "is_client": true, "is_supplier":false}

              1.2 Get contact ID

              Filter e.g. by "company", to get the contact ID. Retrieved contact ID in this case is "2482665622824027155".
              Request
              $ curl -X GET -H "Content-Type: application/json" -H "Authorization: Token ~Your-API-key~" -H "X-Elorus-Organization: ~Your-Organization-ID~" https://api.elorus.com/v1.1/contacts/?company="Elorus SA"

              1. Login to your Elorus account
              2. Go to: "Settings -> Document Types"
              3. Click "+Add" on top right corner
              5. And set it up as the picture bellow. Title used is for example purposes, the field can take any name.
              7. Click "Save"

              2.2 Get document type ID

              Search by "title", to get the document type ID. Retrieved document type ID in this case is "2482671444392150335".
              Request
              $ curl -X GET -H "Content-Type: application/json" -H "Authorization: Token ~Your-API-key~" -H "X-Elorus-Organization: ~Your-Organization-ID~" https://api.elorus.com/v1.1/documenttypes/?search="Απλό Τιμολόγιο"

              3.1 About taxes

              By default there are some taxes preset in Elorus. Usually 24% tax is used for Greece "Invoice" so there is a need for a new tax to be created, use of Elorus' UI is mandatory.
              Steps to create new tax:
              1. Login to your Elorus account
              2. Go to: "Settings -> Taxes"
              3. Click "+Add" on top right corner
              5. And set it up as the picture bellow.
              7. Click "Save"

              3.2 Get taxes ID

              Search by "title", to get the tax ID. Retrieved tax ID in this case is "2482704215621765055".
              Request
              $ curl -X GET -H "Content-Type: application/json" -H "Authorization: Token ~Your-API-key~" -H "X-Elorus-Organization: ~Your-Organization-ID~" https://api.elorus.com/v1.1/taxes/?search="Απλός φόρος"&search_fields=title

              4.1 Create a numbering sequence

              Create a numbering sequence so that the "Απλό Τιμολόγιο" document type has its own numbering.
              Request
              $ curl -X POST -H "Content-Type: application/json" -H "Authorization: Token ~Your-API-key~" -H "X-Elorus-Organization: ~Your-Organization-ID~" https://api.elorus.com/v1.1/numberingsequence/ -d '{"name": "Απλό-Τιμ", "description": "Sequence for -Απλό τιμολόγιο- document type", "documenttype": "2482671444392150335"}'

              4.2 Get numbering sequence ID

              Search by "name", to get the numbering sequence ID. Retrieved numbering sequence ID in this case is "2482706165696300180".
              Request
              $ ccurl -X GET -H "Content-Type: application/json" -H "Authorization: Token ~Your-API-key~" -H "X-Elorus-Organization: ~Your-Organization-ID~" https://api.elorus.com/v1.1/numberingsequence/?search="Απλό-Τιμ"&search_fields=name

              5.1 Create a simple Invoice

              1. X-Elorus-Organization: your organization ID
              2. calculator_mode: sets your values as "initial" or "total", meaning before or after taxes.
                1. When "total" is used, then it is mandatory to use "unit_total" in items.
                2. When "initial" is used, then it is mandatory to use "unit_value" in items.
                3. Depending on the situation, Elorus will calculate the taxes accordingly.
              3. client: Client id - In this case we created "Elorus SA" contact.
              4. draft: true or false. If true the document is considered to be a work in progress.
              5. documenttype: Document Type ID - In this case we used "Απλό Τιμολόγιο" document type.
              6. sequence: Numbering sequence ID
              Request
              $ curl -X POST -H "Content-Type: application/json" -H "Authorization: Token ~Your-API-key~" -H "X-Elorus-Organization: ~Your-Organization-ID~" https://api.elorus.com/v1.1/invoices/ -d '{"calculator_mode": "total", "currency_code": "EUR", "exchange_rate": "1.000000", "client": "2482665622824027155", "date": "2022-5-19", "draft": true, "documenttype": "2482671444392150335", "items": [{"title": "Διάφορες υπηρεσίες", "unit_total": "12.4", "quantity": "1", "taxes": ["2482704215621765055"]}], "sequence":"2482706165696300180"}'
